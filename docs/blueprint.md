# **App Name**: DevTeam Showcase

## Core Features:

- Team Overview Page: A central landing page displaying the team name, project introduction, and a list of team members with their roles and brief descriptions.
- Member Profile Navigation: Functional navigation links on the main page to easily access individual pages for each team member.
- Individual Member Pages: Dedicated pages for each team member to host and showcase their specific interactive functionalities.
- Interactive Mini-Game Module: Integration of a simple, interactive minigame (e.g., Guess the Number) on a member's individual page, demonstrating user input and output.
- Conversion Utility Module: Inclusion of a unit or currency converter on a member's page, requiring user input for calculations and displaying results.
- Calculator Functionality: Implementation of a basic calculator (e.g., BMI, tips, discounts) on a member's page, designed for user data entry and output display.
- Responsive Design: Ensure the website's layout and content adapt seamlessly across various device sizes for an optimal viewing experience.

## Style Guidelines:

- Primary color: A deep, professional blue (#2E66B8) to convey reliability and modern appeal, balancing creativity with a clean aesthetic.
- Background color: A very light, subtle blue-grey (#F6F8FA) providing a clean, spacious canvas that complements the primary color without being distracting.
- Accent color: A vibrant aqua blue (#21CBED) for interactive elements, highlights, and calls to action, offering a lively contrast to the primary hue.
- Body and headline font: 'Inter', a grotesque-style sans-serif for its modern, neutral, and highly readable qualities, suitable for both titles and longer descriptions.
- Use a set of simple, clean, and modern sans-serif line icons to maintain visual consistency and reinforce a contemporary, functional aesthetic.
- Implement a clear, modular grid-based layout for team member sections and individual functionalities, ensuring easy content consumption and a responsive structure across devices.
- Subtle, smooth transition animations for page navigation and hover effects on interactive elements, enhancing the user experience without being overly distracting.
